# Laravel Multi Authentication (User + Admin)

This project demonstrates a **Laravel multi-authentication system** where:

- **User Authentication** is handled by **Laravel Breeze**
- **Admin Authentication** is implemented manually
- Separate guards, sessions, dashboards, and password reset flows are used
